Dec 22, 2016 | Hits

常用命令

1. git

  • git reset –soft id
    把当前本地commit的版本会退到某个版本，本地修改不会丢失

  • git reset –hard id
    本地修改会丢失。

  • git push -u origin master
    把当前本地分支代码推送到远端master分支上
